GAMEPLAYS

  • Map-based or Casual Gameplay
  • The game was configured from external json file.
  • Eat all the food items spawn in the stage.
  • The app will automatically save the user last game progress every complete of the stage.
  • The app has the option to reset saved data (game progress and leaderboards).
  • A stage is cleared or completed if game is not yet over until time is up or eat the required number of food items.

SKILLS

  • Flexible (Wave) - snake head can bump from this body.
  • Hard headed snake (Pill) - snake can destroy the obstacle.

SCORE

  • User will earn stage points only if completed the stage.
  • As the snake length increases the food credit also vary.
  • Snake will update it's length only if completed the stage.
  • As the snake grows the speed increases.

LEADERBOARD

  • Active user will be highlighted from the list to motivate increase in ranking.
  • User ranking will update every complete of the stage.
