# Skillful Snake

Snake game built using **UIKit**

## Featured List

1. Menu
	- Leaderboard
	- Settings
	- About

2. Game
	- New Game / Continue
	- Snake Collision
	- Pause / Play / Restart
	- Timer Enabled
	- Swipe Gesture
	- Configurable

3. Game Levels
	- Level Design via JSON file
	- Sound Effects (Eating, Collision, Background, Gameover)

4. Skills
	- Allow body collision (flexible body)
	- Hard headed snake (can destroy obstacles)
